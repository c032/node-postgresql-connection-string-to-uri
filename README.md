# node-postgresql-keyword-value-to-uri

Convert PostgreSQL's keyword/value connection strings to URI connection
strings.

## Install

```sh
npm install \
  --ignore-scripts \
  --save \
  'github:c032/node-postgresql-keyword-value-to-uri#js'
```
